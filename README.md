# 📰 Hacker News Reader

A sleek, real‑time Hacker News reader with search, pagination, debouncing, and a saved articles feature. Built with the free Algolia HN API — no API key required.

## 🚀 Live Demo
[Click here to try it live!](https://your-username.github.io/hn-reader)

## ✨ Features
- **🔥 Filter by type** — Top, New, Ask, Show, Jobs.
- **🔍 Search stories** — Debounced search by title (400ms delay) to avoid API spam.
- **📄 Pagination** — "Load More" button to fetch additional pages.
- **⭐ Saved articles** — Save your favorite stories to LocalStorage.
- **📊 Rich metadata** — Points, author, comment count, and time ago.
- **🔗 Clickable titles** — Opens the original article or HN thread.
- **💾 Persistent** — Saved articles persist across browser sessions.
- **🌙 Dark theme** — Glass‑morphism UI consistent with my other projects.

## 🛠️ Built With
- HTML5 / CSS3 / Vanilla JS
- [Algolia Hacker News API](https://hn.algolia.com/api) — Free, no‑key search & filtering
- LocalStorage — Saved articles persistence
- Debouncing — Optimized search performance
